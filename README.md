# @amlplugins/langchain-google-vertexai

Native `@langchain/google-vertexai` SDK re-exported under the `@amlplugins` namespace.

> LangChain Google Vertex AI provider — Gemini and Anthropic models via Google Cloud Vertex region endpoints.

## Install

```bash
npm install @amlplugins/langchain-google-vertexai
```

## Use

```ts
import { /* named exports from @langchain/google-vertexai */ } from "@amlplugins/langchain-google-vertexai";
// or
import sdk from "@amlplugins/langchain-google-vertexai";
```

This package additionally re-exports `@langchain/core` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
